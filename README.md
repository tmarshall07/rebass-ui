## Installing
```
yarn add @tannerjs/scheme-ui
```
or
```
npm i @tannerjs/scheme-ui
```

## Storybook
```
yarn storybook
```

### Publishing
Build the packages:
```
cd packages/scheme-ui
yarn run build
```
Build the Storybook static files:
```
yarn run build-storybook
```
Deploy to Github pages:
```
yarn run deploy-storybook
```
